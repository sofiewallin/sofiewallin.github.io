Skills

I always design with a purpose. My style is minimalistic, even conventional and I always start with the users point of view. In the end I want the websites I create to be usable and accessible with a clearly conveyed message. When I code I focus on the same simplicity and accessibility. In every aspect of the websites I create I want usability, whether it be in the browser or in my code base.
